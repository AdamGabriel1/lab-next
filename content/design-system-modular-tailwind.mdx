# Identidade Visual do Lab

O objetivo era criar uma interface que parecesse um **OS (Sistema Operacional)** moderno.

### Princípios de Design
1. **Curvatura Máxima:** Uso de `rounded-[2.5rem]` para cards.
2. **Tipografia de Impacto:** Inter Black Italic para títulos (`tracking-tighter`).
3. **Contraste de Dark Mode:** Uso de tons `#080b14` para evitar o preto puro e melhorar o descanso visual.

> "Um bom Lab não apenas funciona, ele comunica precisão através do design."
